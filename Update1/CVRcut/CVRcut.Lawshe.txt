@title Critical value for CVR, based on Lawshe
@description Calculate critical values for Lawshe’s content validity ratio (CVR), Lawshe's method.
@param num_judges Numeric. Number of judges who gave their ratings.
@param interpolation String. Method for estimate values not appearing in Lawshe's Table (1974). Three interpolation
options: “none” (default), ‘linear’ , ‘splin’.
@param legend Logic. Legend describing the origin of the resulting critical value. Legend values are:
3 (Tabulated value), 2 (Interpolated value), 1 (Non-tabulated value and no interpolation).

CVRcut_Lawshe(num_jueces = 40, interpolacion = "F",legend = T)

@return 
Dos listas de dataframe: `$Output` for numeric results in four columns: "Method", "CritVal"", "Minjudges",
"Notes"; `$Legend` for description of "Notes".

@details
This function is based on the tables generated in Lawshe (1975). 
If the number is tabulated in Lawshe (1975), then `CVRcut.Lawshe` yields the same table value, independent
of the interpolation method. of the interpolation method. Otherwise, interpolation will be used. This
interpolation is for values of `num_judges` outside the tabulated range (n<2 or n>40).
However, this functionality is experimental.

@references

Lawshe, C. H. (1975). A quantitative approach to content validity. Personnel psychology, 28, 563–575.
https://doi.org/10.1111/j.1744-6570.1975.tb01393.x


@seealso
\code{\link{ValCont::CVR}}
\code{\link{ValCont::CVRcut}}

@examples

### Example 1 ----------
CVRcut_Lawshe(num_jueces = 45, interpolacion = "splin",legend = T)

### Example 2 ----------
CVRcut_Lawshe(num_jueces = 45, interpolacion = "none",legend = T)
